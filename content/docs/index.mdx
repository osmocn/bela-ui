title: Introduction
description: Bela UI is a notification UI library for React, shadcn/ui, and Tailwind CSS.
---

Bela UI is still a work in progress. The library is intentionally small today, and more components and examples will be added over time.

Instead of shipping a large component framework, Bela UI provides carefully crafted building blocks for:

- Notifications
- Inboxes
- Notification Centers
- Preferences
- Toasts

All components are distributed as source code, giving you complete control over styling, behavior, and customization.

## Philosophy

- Copy and own the code
- Type-safe by default
- Built for real-world notification systems
- Designed to work with shadcn/ui
- AI-friendly source code

## What's Included

- Notification Item
- Notification List
- Inbox
- Notification Center
- Toast Components

More components will be added over time.
