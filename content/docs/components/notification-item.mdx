## Props

### Notification

| Property | Type | Description |
|----------|------|-------------|
| `id` | `string` | Unique notification identifier. |
| `title` | `string` | Notification title. |
| `body` | `string \| null` | Optional notification content. |
| `actionUrl` | `string \| null` | Optional URL to navigate to when clicked. |
| `readAt` | `string \| null` | Read timestamp. `null` indicates unread. |
| `archivedAt` | `string \| null` | Archive timestamp. |
| `createdAt` | `string` | Notification creation timestamp. |
