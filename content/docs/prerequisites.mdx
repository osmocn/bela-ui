# Prerequisites

Bela UI is built on top of shadcn/ui.

Before using Bela UI, we assume your project already includes:

- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## What Bela UI Does Not Provide

Bela UI is not:

- A component framework
- A notification backend
- A state management library
- A styling system

Instead, Bela UI focuses entirely on reusable notification-related UI components.

## Styling

Components inherit your existing shadcn/ui theme and design tokens.

This means Bela UI automatically adapts to:

- Light mode
- Dark mode
- Custom color palettes
- Custom radius values
- Custom typography

without additional configuration.
